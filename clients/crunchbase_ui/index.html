About this dataset

The nerdosphere dataset is imported montly from the CrunchBase Excel Data Export, which is in turned based on crowdsourced data from CrunchBase.

This graph shows the distribution of venture capital funding rounds by funding series. The x-axis is round size by decile, and the y-axis is the average round size within that decile.

Get the data

go get https://github.com/attic-labs/noms/...
go install $GOROOT/src/github.com/attic-labs/noms/clients/shove
./shove -source-h="https://ds.noms.io" -source-ds="nerdosphere" -sink-ldb="/tmp/nerdosphere" -sink-ds="nerdosphere"
